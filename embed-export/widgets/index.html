Embeddable Chart Widgets

Self-contained interactive charts from the Open Notation project. Each is a single HTML file — drop it on any server and embed with one line of code.

How to embed

  1. Host the .widget.html file on any web server (GitHub Pages, Netlify, S3, your own server)
  2. Copy the embed code below and paste it into your page's HTML
  3. Replace YOUR_URL with the actual URL of the hosted file

Each widget is fully self-contained — no JSON files, no co-hosted dependencies. The only external resource is D3.js loaded from CDN. The iframe auto-resizes to fit the chart content.

Foreign-Born Share vs. Jobs — Large Cities

Arrow scatter plot showing change in foreign-born population share and jobs per 1,000 residents (2015-2023) for MA cities with population over 40,000.
Self-contained 14 KB D3.js 38 cities
Embed code
<iframe src="YOUR_URL/inequality-large.widget.html" style="width:100%;border:none;" height="740"></iframe> <script> window.addEventListener("message", function(e) { if (e.data && e.data["embed-height"]) { document.querySelector('iframe[src*="inequality-large"]') .style.height = e.data["embed-height"] + "px"; } }); </script>

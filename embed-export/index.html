Embeddable Charts

Open Notation — Gateway Cities Economic Analysis
How to embed: Each chart below has a responsive iframe embed code (DataWrapper-style). Copy the code and paste it into your HTML page. The iframe will automatically resize to fit the chart content. Replace YOUR_HOST_URL with the URL where you host these files.

Foreign-Born Share vs. Jobs — Large Cities

Scatter plot with directional arrows showing change in foreign-born population share and jobs per 1,000 residents (2015 to 2023) for larger Gateway Cities.
D3.js CDN data Self-contained
Embed Code
<div style="min-height:500px"> <iframe src="YOUR_HOST_URL/inequality-large.html" scrolling="no" frameborder="0" style="width:0;min-width:100%;border:none;" height="700" id="inequality-large"> </iframe> </div> <script> window.addEventListener("message",function(e){ if(e.data&&e.data["datawrapper-height"]){ var heights=e.data["datawrapper-height"]; for(var k in heights){ var f=document.getElementById("inequality-large"); if(f)f.style.height=heights[k]+"px"; } } }); </script>

Hosting: Upload the entire embed-export/ folder to any static file host (GitHub Pages, Netlify, S3, etc.). Replace YOUR_HOST_URL in the embed codes with your actual base URL.

Self-contained charts (inequality-large, inequality-small, ma-spikes) fetch data from CDN/APIs and need no co-hosted files. Jobs charts (jobs-by-sector, jobs-dashboard) require the JSON/GeoJSON data files to be in the same directory.

Embed pattern follows the DataWrapper responsive iframe approach: the chart reports its height via postMessage, and the parent page's listener adjusts the iframe height accordingly.
